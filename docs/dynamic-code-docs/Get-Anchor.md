# Get-Anchor

<a id="top"></a>
## Description

This script processes a markdown file, adding anchors to headings and generating a Table of Contents (TOC) that links to those anchors. It is designed to be compatible with Bitbucket and GitStash.

## Examples

### Example 1
```powershell
This command will process the default markdown file located at "..\docs\Generic\testing.md", add anchors to its headings, generate a TOC, and save the output to a new file with ".with-toc.md" appended to the original filename. - Reads a markdown file. - Adds anchors to headings (H2 and H3). - Generates a Table of Contents (TOC) with links to the anchors. - Saves the modified content to a new file.
```
