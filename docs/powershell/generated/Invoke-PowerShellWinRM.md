# Invoke-PowerShellWinRM

## Description

Execute a PowerShell script on a remote server via WinRM.

## Parameters

| Parameter | Description |
|-----------|-------------|
| `-Script` | PowerShell script to execute remotely. |
| `-Server` | Remote server hostname or IP. |
| `-Username` | Username for WinRM authentication. |
| `-Password` | Password for WinRM authentication. |
| `-Transport` | WinRM transport (default: NTLM). |
| `-TimeoutSeconds` | Timeout per command in seconds (default: 300). |

## Examples

### Example 1
```powershell
$r = Invoke-PowerShellWinRM -Script 'Get-Process' -Server 'srv01.corp.local' -Username 'admin' -Password 'pass'
```
