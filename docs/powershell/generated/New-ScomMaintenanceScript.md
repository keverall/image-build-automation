# New-ScomMaintenanceScript

## Description

Build a PowerShell script for SCOM maintenance mode start/stop on a group.

## Parameters

| Parameter | Description |
|-----------|-------------|
| `-GroupDisplayName` | SCOM group display name. |
| `-DurationSeconds` | Duration in seconds (used for start). |
| `-Comment` | Maintenance comment string. |
| `-Operation` | 'start' or 'stop' (default: start). |

## Examples

### Example 1
```powershell
$ps = New-ScomMaintenanceScript -GroupDisplayName 'PROD-CLUSTER-01' -DurationSeconds 14400 -Comment 'iRequest'
```
